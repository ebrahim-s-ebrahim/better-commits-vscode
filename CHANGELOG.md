# Changelog

All notable changes to the "Better Commits" extension will be documented in this file.

## [0.1.0] - 2026-03-29

### Added

- Guided commit wizard with multi-step Quick Pick flow
- Commit type selection with branch-based auto-inference
- Commit scope selection with custom scope support
- Ticket/issue extraction from branch name (supports `PROJ-123`, `feat/42`, etc.)
- Commit title input with live max-length validation
- Optional commit body input with period-splitting
- Footer support: breaking changes, deprecations, closes, trailers, custom
- `.better-commits.json` config compatibility with the [better-commits CLI](https://github.com/Everduin94/better-commits)
- SCM title bar button (✏️) for quick access
- Multi-repository workspace support
- Deep-merge config loading (workspace → global → defaults)
